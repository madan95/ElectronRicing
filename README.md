# ElectronJS

Trying out ElectronJS

## Getting Started

Once you have the clone of the repo, start working on app/

### Prerequisites

- node:10.15.3

### Installing

Install modules
```
npm install
```

In a new terminal run watch to start the application
```
npm run start
```
